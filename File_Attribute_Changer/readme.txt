Небольшая утилита для изменения атрибутов файла (папки, группы файлов, группы папок).
Может задавать значение атрибута "системный", который в сочетании с  
атрибутом "скрытый" довольно хорошо может спрятать файл в проводнике от любопытных глаз.

Изменять атрибуты можно и у папок (даже у нескольких одновременно), но тут дело обстоит немного посложнее, так-как в программе используется стандартный диалог выбора файла и папку там выбрать никак нельзя.
Выход из такой ситуации может быть такой: либо прописывать пути к папкам вручную, либо воспользоваться этим доалогом открытия файлов и выбрать любой файл из той папки, атрибуты которой нужно поменять, а потом подтереть имя этого файла, оставив только путь к папке (символ "\", который стоял перед именем файла тоже нужно удалить).

Если работа идет одновременно с несколькими файлами (или папками) их имена нужно разделять символом "|".
